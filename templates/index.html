Este quiz interativo foi desenvolvido como projeto educacional sobre a Venezuela, abordando aspectos geográficos, culturais e históricos do país sul-americano. O sistema apresenta interface responsiva, animações CSS3, validações JavaScript e elementos visuais temáticos baseados na identidade visual venezuelana. Tecnologias utilizadas: HTML5, CSS3, JavaScript ES6, Google Fonts, Font Awesome.

Criado por Natan
